
## How to Run

### Server
In the `server` directory, run the following commands.

#### Setup
```
docker-compose up -d
npm install
```

#### Run
```
npm start
```

### Chat-Client
In the `chat-client` directory, run the following commands.

#### Setup
```
npm install
```

#### Run
```
npm start
```
